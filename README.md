# KCA Student Election System
A secure web-based student election management system built for KCA University.

**Features**
OTP-based voter authentication (email verification)
Candidate and position management
Real-time vote tracking
Election results with graphs
Admin and officer dashboards

**Tech Stack**
PHP
MySQL
PHPMailer (SMTP email)
Bootstrap 5
